HOW TO ADD THE MISSING ARTICLE IMAGES
=====================================
Drop an image into THIS folder (assets/news/) using the exact filename below.
The matching news card will use it automatically the next time the site loads.
No code changes needed.

- Preferred format: .jpg  (also accepted: .png or .jpeg — just keep the same name)
- Recommended size: 1200 x 720 px (landscape). Anything close works; it's cropped to fit.

FILENAMES TO ADD (outlet  ->  filename):
  Bloomberg .................................. bloomberg.jpg
  Business Insider ........................... business-insider.jpg
  Tencent Cloud .............................. tencent-cloud.jpg
  VentureBeat (Telegram / TON) ............... venturebeat-telegram-ton.jpg
  VentureBeat (Web3 platform for superapps) .. venturebeat-superapps.jpg
  The Street (attention economy) ............. thestreet-mini-everything.jpg
  The Street (seed funding) .................. thestreet-seed.jpg
  BlockTelegraph (global expansion) .......... blocktelegraph-expansion.jpg
  BlockTelegraph (seed round) ................ blocktelegraph-seed.jpg
  Coruzant ................................... coruzant.jpg
  OKX ........................................ okx.jpg

Until a file is added, that card just shows its branded colour cover (nothing breaks).
After adding files, commit & push (or tell me) and they go live.
