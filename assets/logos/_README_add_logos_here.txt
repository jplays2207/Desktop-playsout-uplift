HOW TO ADD THE "FEATURED IN" LOGOS
==================================
Drop a logo file into THIS folder (assets/logos/) using the exact filename below.
The moving logo strip on the homepage will use it automatically on the next load.

FORMAT (important):
- Best: .svg  (sharpest). Also works: .png
- Use a WHITE or LIGHT version on a TRANSPARENT background
  (the strip sits on a dark navy background, so dark logos won't show).
- Any width is fine; it's scaled to ~40px tall.

FILENAMES TO ADD (outlet -> filename):
  Bloomberg ................ bloomberg.svg   (or bloomberg.png)
  Business Insider ......... businessinsider.svg
  Cointelegraph ............ cointelegraph.svg
  Decrypt .................. decrypt.svg
  GamesBeat ................ gamesbeat.svg
  VentureBeat .............. venturebeat.svg
  HackerNoon ............... hackernoon.svg
  The Defiant .............. thedefiant.svg
  Business Times (IBTimes).. ibtimes.svg
  Tencent .................. tencent.svg
  Esports Insider .......... esportsinsider.svg
  Coruzant ................. coruzant.svg
  The Street ............... thestreet.svg
  Grit Daily ............... gritdaily.svg
  Financial Tech Times ..... financialtechtimes.svg
  OKX ...................... okx.svg
  BlockTelegraph ........... blocktelegraph.svg

Until a logo file is added, that slot shows the outlet NAME as text (so the strip
always looks complete). Add files anytime, then commit & push (or tell me).

TIP: a quick way to get a clean white logo is to search "<outlet> logo svg" or grab it
from the outlet's own press/brand page, then recolor to white if needed.
