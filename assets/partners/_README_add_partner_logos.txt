HOW TO ADD THE "PARTNERS" LOGOS
===============================
Drop a logo file into THIS folder (assets/partners/) using the exact filename below.
The moving partners strip on the homepage picks it up automatically on next load.

FORMAT:
- Best: .svg  (also works: .png)
- Use a WHITE / LIGHT logo on a TRANSPARENT background (strip is on dark navy).
- Any width; scaled to ~40px tall.

FILENAMES (partner -> file):
  KBW Ventures ......... kbw.svg
  Oak Grove ............ oak-grove.svg
  OKX .................. okx.svg
  Kenetic .............. kenetic.svg
  Aptos ................ aptos.svg
  Helika ............... helika.svg
  viu .................. viu.svg
  LINE ................. line.svg
  Young Platform ....... young-platform.svg
  VentureSouq .......... venturesouq.svg

Until a file is added, that slot shows the partner NAME as text, so the strip
always looks complete. Add files anytime, then push (or tell me).
